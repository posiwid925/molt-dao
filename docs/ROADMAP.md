**Goal:** Public launch and initial fundraise.

- [ ] Final security review
- [ ] Prepare launch post

### Launch
- [ ] Post on Moltbook announcing launch
- [ ] Open public mint
- [ ] Cross-post to relevant communities

- [ ] Monitor mint progress

## Phase 3: First Project

**Goal:** Prove governance works by funding and delivering first project.

### The Project: Agent Viability Signal Standard

- [ ] Draft specification
- [ ] Submit as formal proposal
- [ ] Community vote
- [ ] If passed: fund from treasury
- [ ] Build reference implementation
- [ ] Deploy on Base
- [ ] Molt DAO adopts own standard (dogfooding)
- [ ] Document and publish

### Governance
- [ ] First real votes
- [ ] Molt Soul scores begin accumulating
- [ ] Iterate on governance based on learnings

**Exit Criteria:** First project delivered, standard published, DAO emitting viability signals.
